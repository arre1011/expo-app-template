## 1. Overview & Goals

### Strategy Summary

We use **micro-influencers (<10K followers)** in the sober curious, wellness, fitness, and mental health niches. Our model is **commission-based (no upfront cost)** with a "Gifted + Commission" approach: free Premium access + recurring revenue share.

### Why Micro-Influencers?

| Metric | Micro (<10K) | Macro (100K+) | Source |
|--------|-------------|---------------|--------|
| Engagement Rate (Instagram) | 3.86-7% | 1-2% | Stack Influence 2025 |
| Engagement Rate (TikTok) | 10-18% | 1-3% | Stack Influence 2025 |
| Conversion Rate per Engagement | **6.7x higher** | Baseline | Stack Influence 2025 |
| Healthcare Conversion | **22.2x higher** | Baseline | iQfluence 2026 |
| Cost per Engagement | $0.20 | $0.33 | AMA 2024 |
| Cost per Subscriber (estimated) | $10-20 | $50-100+ | Derived |
| Commission-only Acceptance | Higher (portfolio-building) | Very low (94% want flat fee) | SARAL, The Motherhood |

### Goals by Phase

- **Phase 1 (Month 1-6):** 20-50 active partners, validate conversion rates
- **Phase 2 (Month 6-12):** 50-200 active partners, optimize commission structure
- **Phase 3 (Month 12-24):** 200+ partners, self-service portal, automation

## 4. Outreach Process

### Channel Priority

1. **Instagram DM** (highest response rate for micro-influencers)
2. **Email** (if available in bio)
3. **TikTok comment → DM** (lower response rate but high-value creators)

### Response Rate Benchmarks

| Approach | Expected Response Rate |
|----------|----------------------|
| Generic mass DM | 4-5% |
| Personalized email | ~18% |
| Well-personalized influencer outreach | 30-40% |
| Highly curated + personalized | Up to 90% |

**Key: Personalization doubles response rates.** Always reference specific content.

### Outreach Strategy: Product-Led + Monetization Hook

The first message has **two hooks**:
1. **Immediate value:** A free 1-month trial so they can test the app right away
2. **Monetary interest:** The 30% recurring commission via the partner landing page

**Why give a trial coupon upfront?**
- If the app is good, the product sells itself — authentic content comes from real usage
- A 1-month trial costs us nothing (no lost revenue from a non-customer)
- Creates reciprocity — we give before we ask
- Acts as screening: if they don't even try the app, they wouldn't be a good partner
- Reduces friction: no commitment needed, just "try it"

```

and really loved your post about [specific post].
```

### Two-Code System

|------|---------|------------|------------|
| `[NAME]-TRIAL` | Personal trial for the influencer (1 month free) | DM 1 (first contact) | 6 months |

**Important:** The trial code is a separate Apple/Google offer code. Track both codes in the spreadsheet.

### Optional: 5 Friend Codes (Social Amplification)

Instead of giving the influencer just 1 personal trial code, give them **5 codes** to share with friends and family.

**Why this works:**
- **Drinking is social** — people rarely drink alone. When friends track together, the app becomes part of the social experience
- **Group accountability** — friends motivate each other to stay mindful, increasing retention
- **Organic content opportunities** — group photos, Stories, shared experiences ("We all tracked last night!") create authentic content the influencer wouldn't generate alone
- **Word-of-mouth amplification** — 5 people talking about the app in their social circles multiplies reach exponentially
- **Zero cost** — trial codes cost nothing (these people weren't paying customers)
- **Screening signal** — if the influencer actually shares all 5 codes, they're engaged and social — exactly the partner profile we want
- **Content ideas emerge naturally** — friends comparing BAC estimates, group challenges, morning-after check-ins

**Why 5, not more?**
- 5 is enough for a close friend group (dinner, night out)
- Not so many that codes feel cheap or disposable
- Creates exclusivity ("I have a few codes left")
- Manageable to track per influencer

**Tracking:** Add columns `Friend Codes Given` and `Friend Codes Redeemed` to the spreadsheet. Track how many of the 5 get used — this is an early signal for partner quality.

## 5. Onboarding Flow

### The Journey

```
1. You send DM 1 with personal trial code + link to partners.html
     ↓
2. Influencer downloads app, tries it for free (1 month trial)
     ↓
3. Influencer replies "Yes, I'm interested in the partner program"
     ↓
4. You generate their partner code ([NAME]-GC) and send DM 2:
   Short message + link to onboarding page:
   "glasscount.com/partners/welcome?code=SARAH-GC&name=Sarah"
     ↓
5. Onboarding page shows:
   - Their personalized partner code with copy button
   - Shareable follower link (glasscount.com/go?code=SARAH-GC&ref=Sarah)
   - FTC disclosure templates (copy-paste ready)
   - Content ideas (3 levels)
   - Do's and Don'ts
   - FAQ
     ↓
6. Influencer shares link → Followers land on go.html →
   Download app → Enter code → 7 days free trial
     ↓
7. GoMarketMe/RevenueCat tracks conversions
     ↓
8. Monthly payout email with earnings summary
```

### The Three Pages

| Page | URL | Audience | Purpose |
|------|-----|----------|---------|
| **partners.html** | glasscount.com/partners | Potential partners (DM 1) | Recruitment — shows commission, how it works, science |
| **partners/welcome.html** | glasscount.com/partners/welcome?code=X&name=Y | Confirmed partners (DM 2) | Onboarding — code, shareable link, disclosure, content ideas |
| **go.html** | glasscount.com/go?code=X&ref=Y | Followers | Conversion — download app + enter code |

### Key Principle: Keep Messages SHORT

All detailed information lives on the web pages, not in messages. Messages are max 3-5 sentences + one link.

**DM 1** → links to `partners.html` (recruitment)
**DM 2** → links to `partners/welcome.html?code=...&name=...` (onboarding)
**Influencer posts** → links to `go.html?code=...&ref=...` (follower conversion)

## 7. Content Guidelines (Structured Freedom)

### Philosophy: "Structured Freedom"

Research consistently shows that the optimal approach is providing clear guardrails
but letting influencers control the creative execution.

**Key findings:**
- 52% of consumers find influencer content more compelling than scripted ads (Statista)
- Over-controlled content reduces engagement by weakening parasocial relationships (Journal of Business Research)
- Briefs with >3-4 talking points feel forced and inorganic (Aspire)
- Long-term partnerships generate 300% more engagement than one-off campaigns (SARAL)

### What We Control (Non-Negotiables)

1. **FTC Disclosure:** Every post MUST include #Ad at the beginning
2. **Health Disclaimer:** MUST mention BAC values are estimates
3. **Prohibited Claims:**
   - ❌ NEVER say the app determines if you're safe to drive
   - ❌ NEVER make medical or diagnostic claims
   - ❌ NEVER suggest the app replaces professional medical advice
   - ❌ NEVER promote excessive drinking
4. **Required Talking Points (max 2-3):**
   - The app helps track drinks mindfully
   - All data stays on your device (privacy)
   - Estimated alcohol level calculation helps awareness

### What Influencers Control (Full Freedom)

- Script / narrative structure
- Filming style and editing
- Tone of voice and personality
- Format (Reel, TikTok, Story, static post, YouTube)
- Posting timing (within a general window)
- Whether to use our suggestions or their own concept

### Content Ideas (Shared on Landing Page)

**Level 1 -- Quick & Easy (~5 min effort)**
Quick Story/clip showing the app. Mention you're trying a mindful drinking app,
show the home screen with chart and logged drinks. Low barrier to entry.

**Level 2 -- Authentic Review (~15 min effort)**
Use the app for a few days, share honest thoughts in a Reel/TikTok.
What surprised you? What did you learn? Be genuine -- even about improvements.

**Level 3 -- Story-Driven (~30+ min filming)**
Document a night out: log each drink, show estimated level rising, stay within
your limit. Next day: show the positive outcome (gym, work, productive morning).
Narrative: "Because I tracked, I stayed mindful, and today I feel great."

### Content Strategy Recommendation for Influencers

**Reels/TikTok for Discovery + Stories for Conversion:**
- Reels: 2.08% engagement rate (highest on Instagram), best for reach
- Stories: Direct link/code sharing, drives actual installs
- Best combo: Viral Reel → follow-up Stories with code and CTA

### Authenticity Guidelines

- "Share your honest experience -- including things that could be better"
- Genuine reviews (with nuance) convert better than pure praise
- 61% of consumers trust influencer recommendations when they feel genuine
- Audiences in health/wellness are especially sensitive to inauthenticity

## 8. Legal & Compliance

### FTC Requirements (United States)

| Requirement | Details |
|-------------|---------|
| **Disclosure** | #Ad at the START of every caption/post |
| **Video** | Verbal disclosure in first few seconds |
| **Stories** | Text overlay on EACH story frame + Paid Partnership tag |
| **Affiliate links** | "This is an affiliate link" near the link |
| **Penalty** | Up to $53,088 per violation |

**Copy-paste disclosure templates (provide to every influencer):**

Instagram/TikTok caption:
```
#Ad | I partnered with @glasscount. I earn a commission when you use my code [CODE].
```

Video (spoken):
```
"This video is in partnership with GlassCount. I earn a commission if you use my code."
```

Stories:
```
Use "Paid Partnership" tag AND write "#Ad" visibly on screen.
```

### UK ASA Requirements

- #Ad at the START of caption (not buried)
- "Ad - Gifted" for gifted-only partnerships
- ASA uses AI monitoring (57% compliance rate in 2025)

### Germany (for future DE expansion)

- Must use "#Werbung" or "#Anzeige" (not just English #Ad)
- German case law requires German-language disclosure for German audiences

### Health-Specific Compliance

**Our app MUST NEVER be positioned as:**
- A tool to determine driving fitness
- A medical device or diagnostic tool
- A replacement for professional medical advice
- Encouragement to drink alcohol

**Required health disclaimer in all content:**
> "Alcohol level values are estimates only based on the Widmark formula.
> They must NOT be used to determine fitness to drive."

### Does Disclosure Hurt Conversion?

**No.** Research shows:
- Content WITH disclosure has HIGHER engagement than undisclosed affiliate content (Springer/ISR 2024)
- Transparency builds long-term trust
- 75% of consumers accept sponsored content when it's honest
- Non-disclosure risk ($53K/violation) far outweighs any marginal engagement cost

## 9b. Deep Linking & App Onboarding Strategy

### Why Deep Linking Matters

| Metric | Without Deep Linking | With Deep Linking | Source |
|--------|---------------------|-------------------|--------|
| Click-to-install rate | ~5% | 30%+ | Branch |
| Conversion improvement | Baseline | +52% | Branch / pimms.io |
| Onboarding completion | 40-50% | 92% (Lark Health) | Branch case study |
| Manual code redemption drop-off | 20-35% abandon | 0% (auto-applied) | Industry data |

### How Deferred Deep Linking Works

```
1. Follower clicks influencer link (e.g., glasscount.link/SARAH-GC)
     ↓
2. Branch SDK detects: app NOT installed
     ↓
3. User redirected to App Store / Google Play automatically
     ↓
4. User installs app & opens it
     ↓
5. On first launch, Branch SDK queries server:
   "Was there a deferred deep link for this device?"
     ↓
6. Server returns: { code: "SARAH-GC", ref: "Sarah" }
     ↓
7. App reads parameters → auto-applies promo code on paywall
```

**The user clicks the link ONCE. No second click needed.**
Branch uses device fingerprinting (+ NativeLink on iOS) for near-100% match accuracy.

### App Onboarding: Key Research Findings

**Drop-off per step:**

| Onboarding Steps | Users Remaining | Source |
|-----------------|----------------|--------|
| 1 | 100% | |
| 2 | ~80% | |
| 3 | ~64% | Gabor Cselle |
| 4 | ~51% | Appcues |
| 5 | ~41% (cliff!) | Appcues |

- **82% of trial starts happen on Day 0** — users are most motivated right after install (RevenueCat)
- **Referred users have +37% retention, +16% LTV** vs. organic users
- Moving paywall earlier: 5x revenue (Rootd), 20% more subs (Billdu)

**"Coupon Field Anxiety" (Baymard Institute):**
- A visible empty promo code field makes users WITHOUT codes feel they're overpaying
- Some leave to search for codes and never return
- **Best practice:** Hide behind collapsed "Have a promo code?" link

### Technical Implementation (Phase 2)

**App integration (`_layout.tsx`):**
```typescript
import DeepLinkNow from '@deeplinknow/react-native';

// Initialize in root component
DeepLinkNow.initialize('your-api-key');

useEffect(() => {
  // Listen for deep links (works for both deferred and direct)
  const unsub = DeepLinkNow.addDeepLinkListener((data) => {
    const code = data.parameters?.code;   // "SARAH-GC"
    const ref = data.parameters?.ref;     // "Sarah"

    if (code) {
      // Store in Zustand → onboarding detects referred user
      useAppStore.setState({ referralCode: code, referrerName: ref });
    }
  });
  return () => unsub();
}, []);
```

**Onboarding routing (based on referral state):**
```typescript
const referralCode = useAppStore(s => s.referralCode);
const referrerName = useAppStore(s => s.referrerName);

if (referralCode) {
  // → Flow B: 2-screen referred onboarding (see below)
} else {
  // → Flow A: Standard 4-5 screen onboarding
}
```

**Landing page integration (`go.html`):**
Once deep links are set up, replace the two Store buttons with a single smart link:
```html

<!-- Phase 2 (deep links): One smart button -->
<a href="https://glasscount.dlnow.link/SARAH-GC" class="download-btn">
```

DeepLinkNow auto-detects the device and redirects to the correct store. The referral data is carried through the install process automatically.
