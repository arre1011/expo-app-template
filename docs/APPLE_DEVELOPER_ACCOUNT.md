# Apple Developer Account erstellen

Schritt-für-Schritt Anleitung zum Erstellen eines Apple Developer Accounts für den App Store.

## 💰 Kosten

**Apple Developer Program:**
- **€99 pro Jahr** (Pflicht für App Store Veröffentlichung)
- Zahlung per Kreditkarte, Debitkarte oder PayPal
- Automatische Verlängerung (kann deaktiviert werden)
- Keine Setup-Gebühren, keine versteckten Kosten

**Was ist inklusive:**
✅ App Store Veröffentlichung (iOS, iPadOS, macOS, watchOS, tvOS)
✅ TestFlight für Beta-Testing (bis zu 10.000 externe Tester)
✅ App Store Connect Zugang
✅ Code-Signing Zertifikate
✅ Push Notifications
✅ In-App Purchases / Subscriptions
✅ Developer Support & Ressourcen

---

## 📋 Voraussetzungen

**Du brauchst:**
- [ ] **Apple ID** (kann neu erstellt werden oder bestehende nutzen)
- [ ] **Kreditkarte oder Debitkarte** (VISA, Mastercard, Amex)
- [ ] **Telefonnummer** für Zwei-Faktor-Authentifizierung (2FA)
- [ ] **Vollständiger Name & Adresse** (wie auf Ausweis/Rechnung)
- [ ] **D-U-N-S Nummer** (nur für Firmen, siehe unten)

**Wichtig:**
- Du musst **mindestens 18 Jahre alt** sein
- Die Registrierung dauert ca. **15-30 Minuten**
- Account-Freischaltung kann **24-48 Stunden** dauern

## 🚀 Schritt-für-Schritt Anleitung

### Schritt 1: Apple ID vorbereiten

**Option A: Bestehende Apple ID nutzen**

Falls du bereits eine Apple ID hast (z.B. für iCloud, App Store):

1. Gehe zu [appleid.apple.com](https://appleid.apple.com)
2. Login mit deiner Apple ID
3. **Zwei-Faktor-Authentifizierung aktivieren:**
   - Sicherheit → Zwei-Faktor-Authentifizierung → Aktivieren
   - Füge Telefonnummer hinzu
   - Bestätige per SMS-Code
4. **Zahlungsmethode hinzufügen:**
   - Zahlung und Versand → Zahlungsmethode hinzufügen
   - Kreditkarte oder Debitkarte eingeben

**Option B: Neue Apple ID erstellen**

Falls du eine dedizierte Developer-Apple-ID willst:

1. Gehe zu [appleid.apple.com/account](https://appleid.apple.com/account)
2. Klicke "Deine Apple ID erstellen"
3. Fülle Formular aus:
   ```
   Vorname: [Dein Vorname]
   Nachname: [Dein Nachname]
   Land/Region: Deutschland
   Geburtsdatum: [Dein Geburtsdatum]
   E-Mail: developer@deinedomain.com (oder beliebig)
   Passwort: [Starkes Passwort]
   Telefonnummer: +49 xxx xxxxx
   ```
4. Bestätige Email-Adresse (Code per Email)
5. Bestätige Telefonnummer (Code per SMS)
6. Aktiviere Zwei-Faktor-Authentifizierung

**Empfehlung:**
- Nutze bestehende Apple ID, wenn es deine persönliche ist
- Erstelle neue Apple ID, wenn du professionell/geschäftlich entwickelst

---

### Schritt 2: Apple Developer Program beitreten

1. **Gehe zu Apple Developer Website:**
   - [developer.apple.com/programs/enroll](https://developer.apple.com/programs/enroll)

2. **Klicke "Start Your Enrollment"**

3. **Login mit Apple ID:**
   - Nutze die vorbereitete Apple ID
   - Zwei-Faktor-Code eingeben (per SMS/App)

4. **Account-Typ wählen:**

   **Option A: Individual (Einzelperson)**
   ```
   Für dich geeignet, wenn:
   ✅ Du als Privatperson entwickelst
   ✅ Du keine Firma hast
   ✅ App läuft unter deinem Namen im App Store

   Vorteile:
   • Schnellere Registrierung (keine D-U-N-S nötig)
   • Einfacher Setup
   • Sofort loslegen

   App Store Anzeige:
   "Developer: [Dein Name]"
   ```

   **Option B: Organization (Firma/Organisation)**
   ```
   Für dich geeignet, wenn:
   ✅ Du eine eingetragene Firma hast (GmbH, UG, etc.)
   ✅ App soll unter Firmennamen laufen
   ✅ Mehrere Team-Mitglieder entwickeln

   Voraussetzung:
   • D-U-N-S Nummer (kostenlos, siehe unten)
   • Handelsregisterauszug
   • Geschäftsführer-Berechtigung

   App Store Anzeige:
   "Developer: [Firmenname]"
   ```

   **Empfehlung für Einsteiger:**
   - Starte mit **Individual**
   - Upgrade zu Organization später möglich (wenn nötig)
   - Individual reicht für 99% der Solo-Entwickler

5. **Persönliche Daten eingeben:**
   ```
   Legal Name: [Dein vollständiger Name wie auf Ausweis]
   Address: [Deine Adresse]
   City: [Stadt]
   Postal Code: [PLZ]
   Country: Deutschland
   Phone: +49 xxx xxxxx
   ```

6. **Entwickler-Vereinbarung lesen & akzeptieren:**
   - Apple Developer Agreement
   - Checkbox anhaken
   - "Continue" klicken

7. **Zahlung:**
   - Betrag: **€99.00** (jährlich)
   - Zahlungsmethode auswählen:
     - Kreditkarte (empfohlen)
     - Debitkarte
     - PayPal (in manchen Regionen)
   - Kartendaten eingeben
   - **"Purchase" bestätigen**

8. **Bestätigung:**
   - Du erhältst Email: "Welcome to the Apple Developer Program"
   - Receipt für €99 Zahlung

---

### Schritt 3: Account-Aktivierung abwarten

**Normale Aktivierungszeit:**
- **Sofort bis 48 Stunden** (meist innerhalb 24h)
- Bei Individual: Oft sofort
- Bei Organization: Kann länger dauern (Verifizierung)

**Was passiert während der Aktivierung:**
1. Apple verifiziert deine Daten
2. Zahlung wird geprüft
3. Account wird freigeschaltet

**Du erhältst Email:**
```
Subject: Your enrollment in the Apple Developer Program is complete

Your Apple Developer Program membership is now active.
You can now access the full range of development resources.
```

**Während du wartest:**
- Entwickle deine App weiter
- Erstelle Icons & Screenshots
- Schreibe Privacy Policy
- Lies App Store Review Guidelines

---

### Schritt 4: App Store Connect Zugang

**Nach Aktivierung:**

1. **Gehe zu App Store Connect:**
   - [appstoreconnect.apple.com](https://appstoreconnect.apple.com)

2. **Login mit Apple ID:**
   - Nutze dieselbe Apple ID wie für Developer Account
   - Zwei-Faktor-Code eingeben

3. **Erste Schritte:**
   - Agreements, Tax, and Banking
   - Paid Applications Agreement akzeptieren (für In-App Purchases)

4. **Team-Einstellungen:**
   - Users and Access → Dein Account
   - Role: Account Holder (Admin)
   - Access zu allen Features

## 🏢 Für Firmen: D-U-N-S Nummer

**Nur nötig für Organization Account!**

### Was ist eine D-U-N-S Nummer?

- Eindeutige 9-stellige Unternehmenskennung
- Vergeben von Dun & Bradstreet
- **Kostenlos** (wenn über Apple beantragt)
- Weltweit anerkannt

### D-U-N-S Nummer beantragen:

1. **Gehe zu Apple D-U-N-S Lookup:**
   - [developer.apple.com/enroll/duns-lookup](https://developer.apple.com/enroll/duns-lookup)

2. **Firmen-Daten eingeben:**
   ```
   Legal Entity Name: [Firmenname laut Handelsregister]
   Headquarters Address: [Firmenadresse]
   Mailing Address: [Postadresse]
   Phone Number: [Firmen-Telefon]
   Website: [Firmen-Website] (optional)

   Your Contact Information:
   Name: [Dein Name]
   Position: [Geschäftsführer / CEO]
   Email: [Deine Email]
   Phone: [Deine Telefon]
   ```

3. **Submit Request:**
   - Apple leitet Antrag an Dun & Bradstreet weiter

4. **Wartezeit:**
   - **1-5 Werktage** (meist 2-3 Tage)
   - Du erhältst Email mit D-U-N-S Nummer

5. **D-U-N-S in Developer Enrollment eingeben:**
   - Zurück zu developer.apple.com/programs/enroll
   - "Organization" wählen
   - D-U-N-S Nummer eingeben

## 💳 Zahlungseinstellungen (für Subscriptions)

**Wichtig für In-App Purchases!**

Nach Account-Aktivierung:

### 1. Agreements akzeptieren:

**App Store Connect → Agreements, Tax, and Banking**

**Paid Applications Agreement:**
```
Status: Active (muss akzeptiert sein)

Was das bedeutet:
✅ Du kannst kostenpflichtige Apps verkaufen
✅ In-App Purchases / Subscriptions aktiviert
✅ Apple behält 30% (15% nach Jahr 1 bei Subscriptions)
```

### 2. Steuerinformationen:

**Tax Forms:**

**Für Deutschland:**
```
Tax Residency: Deutschland
Steuerstatus: Individual (Privatperson) oder Business (Firma)

Für Individual:
- Name
- Adresse
- Steuer-ID (optional, empfohlen)

Für Business:
- Firmenname
- USt-IdNr. (falls vorhanden)
- Handelsregister-Nr.
```

**W-8BEN Formular (USA):**
- Da Apple US-Firma ist
- Bestätigt, dass du NICHT in USA steuerpflichtig bist
- Verhindert US-Steuerabzug (30%)
- Wird automatisch generiert

### 3. Banking (Auszahlungen):

**Bank Account Information:**

```
Kontoinhaber: [Dein Name / Firmenname]
IBAN: DExx xxxx xxxx xxxx xxxx xx
BIC/SWIFT: [Deine BIC]
Bank Name: [Name deiner Bank]
Bank Address: [Adresse der Bank]
```

**Auszahlungsrhythmus:**
- Monatlich (wenn > €150 Umsatz)
- 45 Tage nach Monatsende
- Direkt auf dein Bankkonto

**Provision:**
- Apple behält **30%** bei erstem Jahr
- **15%** bei Subscriptions ab Jahr 2 (Small Business Program)
- Du erhältst 70% bzw. 85%

## ⚠️ Häufige Probleme

### Problem 1: Zahlung wird abgelehnt

**Ursachen:**
- Karte hat kein Guthaben
- 3D-Secure nicht aktiviert
- Internationale Zahlungen blockiert

**Lösung:**
1. Kontaktiere deine Bank
2. Aktiviere internationale Online-Zahlungen
3. Nutze alternative Zahlungsmethode
4. Versuche andere Kreditkarte

### Problem 2: Account wird nicht aktiviert

**Wenn nach 48h nichts passiert:**

1. **Check Email:**
   - Spam-Ordner prüfen
   - Email von @apple.com oder @email.apple.com

2. **Login zu Developer Account:**
   - [developer.apple.com/account](https://developer.apple.com/account)
   - Status prüfen: "Membership" sollte "Active" sein

3. **Apple Support kontaktieren:**
   - [developer.apple.com/contact](https://developer.apple.com/contact)
   - "Membership and Account" wählen
   - Deine Apple ID & Zahlungsbestätigung bereithalten

### Problem 3: Zwei-Faktor-Authentifizierung Probleme

**Wenn SMS nicht ankommt:**

1. **Alternatives Gerät:**
   - Apple ID → Sicherheit → Vertrauenswürdige Geräte
   - Füge iPhone/iPad hinzu

2. **Telefonnummer ändern:**
   - appleid.apple.com → Sicherheit
   - Telefonnummer aktualisieren

3. **Recovery Key nutzen:**
   - Falls eingerichtet
   - appleid.apple.com → Account-Wiederherstellung

### Problem 4: Name stimmt nicht überein

**Fehlermeldung: "Name doesn't match credit card"**

**Lösung:**
- Nutze exakt denselben Namen wie auf Kreditkarte
- Keine Abkürzungen
- Umlaute: ä = ae, ö = oe, ü = ue (oder original)
- Beispiel: "René Müller" oder "Rene Mueller"

## 🔄 Account-Verlängerung

**Automatische Verlängerung:**
- Account verlängert sich automatisch nach 1 Jahr
- €99 werden von hinterlegter Zahlungsmethode abgebucht
- 30 Tage vor Ablauf erhältst du Erinnerungs-Email

**Automatische Verlängerung deaktivieren:**
1. developer.apple.com/account
2. Membership → Auto-Renewal Settings
3. "Turn Off Auto-Renewal"

**Wichtig:**
- Wenn Account abläuft: Apps bleiben im App Store
- Aber: Keine Updates mehr möglich
- Subscriptions laufen weiter, aber keine neuen möglich

## 💡 Pro Tips

1. **Nutze bestehende Apple ID:**
   - Einfacher als neue zu erstellen
   - Dieselbe ID für Developer & App Store Connect

2. **Individual vs Organization:**
   - Start mit Individual (schneller, einfacher)
   - Upgrade zu Organization später wenn nötig
   - Migration möglich (mit Support-Ticket)

3. **Calendar Reminder:**
   - Setze Reminder 1 Monat vor Verlängerung
   - Prüfe ob Auto-Renewal aktiv ist
   - Budget für €99/Jahr einplanen

4. **Backup Codes:**
   - Zwei-Faktor Recovery Key generieren
   - Sicher aufbewahren (nicht digital!)
   - Verhindert Account-Lockout

5. **Separate Email:**
   - Nutze dedizierte Email für Developer Account
   - Z.B. dev@deinedomain.com
   - Einfacher bei Team-Übergaben
